Exercice 1

Montrons par récurrence que pour des nombres réels \(x_1, x_2, \ldots, x_n\) et \(y_1, y_2, \ldots, y_n\) tels que

  1. \(\forall i \in \{1, 2, 3, \ldots, n\}, x_i \leq y_i\) alors \[\sum_{i=1}^{n} x_i \leq \sum_{i=1}^{n} y_i.\]

  2. \(\forall i \in \{1, 2, 3, \ldots, n\}, x_i \leq y_i\) alors \[\prod_{i=1} x_i \leq \prod_{i=1} y_i.\]

  3. \(\forall i \in \{1, 2, 3, \ldots, n\}, x_i \leq y_i\) et \(\exists i_0 \in \{1, 2, 3, \ldots, n\}, x_{i_0} < y_{i_0}\) alors \[\sum_{i=1}^{n} x_i < \sum_{i=1}^{n} y_i.\]

  4. \(\forall i \in \{1, 2, 3, \ldots, n\}, x_i \leq y_i\) et \(\sum_{i=1}^{n} x_i = \sum_{i=1}^{n} y_i\) alors \(x_i = y_i\), \(\forall i \in \{1, 2, 3, \ldots, n\}\).

Corrigé

(a)

Montrons par récurrence sur \(n\) que \(\sum_{i=1}^{n} x_i \leq \sum_{i=1}^{n} y_i\).

  • Initialement : Pour \(n = 1\), \(x_1 \leq y_1\) donc \(P(1)\) est vraie.
  • Supposons que \(P(k)\) est vraie et montrons que \(P(k+1)\) est vraie.

Supposons que \(\forall i \in \{1, 2, \ldots, k+1\}, x_i \leq y_i\). On en déduit que \(x_{k+1} \leq y_{k+1}\). Sachant que \(\sum_{i=1}^{k} x_i \leq \sum_{i=1}^{k} y_i\), alors \[\sum_{i=1}^{k} y_i - \sum_{i=1}^{k} x_i \in \mathbb{P} \cup \{0\}\]

Ainsi, \[\sum_{i=1}^{k} y_i + y_{k+1} - \left( \sum_{i=1}^{k} x_i + x_{k+1} \right) \in \mathbb{P} \cup \{0\}\]

Par conséquent, \[\sum_{i=1}^{k+1} y_i - \sum_{i=1}^{k+1} x_i \in \mathbb{P} \cup \{0\}\]

Donc, \[\sum_{i=1}^{k+1} y_i > \sum_{i=1}^{k+1} x_i\]

(b)

Montrons par récurrence sur \(n\) que \(\prod_{i=1} x_i \leq \prod_{i=1} y_i\).

  • Initialement : Pour \(n = 1\), \(x_1 \leq y_1\) donc \(P(1)\) est vraie.
  • Supposons que \(P(k)\) est vraie et montrons que \(P(k+1)\) est vraie.

Supposons que \(\forall i \in \{1, 2, \ldots, k+1\}, 0 \leq x_i \leq y_i\). On a : \[y_{k+1} - x_{k+1} \in \mathbb{P} \cup \{0\}\] \[\prod_{i=1}^{k} x_i \in \mathbb{P} \cup \{0\} \Rightarrow y_{k+1} \prod_{i=1}^{k} x_i - x_{k+1} \prod_{i=1}^{k} x_i \in \mathbb{P} \cup \{0\}\]

Ainsi, \[\prod_{i=1}^{k+1} y_i - \prod_{i=1}^{k+1} x_i \in \mathbb{P} \cup \{0\}\]

Donc, \[\prod_{i=1}^{k+1} x_i \leq \prod_{i=1}^{k+1} y_i\]

(c)

Pour \(n = 1\), \(x_1 \leq y_1\) et \(x_1 < y_1\) alors \(x_1 < y_1\) donc \(P(1)\) est vraie. Pour \(n = 2\), si \(x_1 \leq y_1\) et \(x_2 \leq y_2\), alors \(y_2 + y_1 > x_2 + x_1\), donc \(P(2)\) est vraie.

Supposons que \(P(k)\) est vraie et montrons que \(P(k+1)\) est vraie.

Supposons que \(\forall i \in \{1, 2, \ldots, k+1\}, x_i \leq y_i\) et \(\exists i_0 \in \{1, 2, \ldots, k+1\}, x_{i_0} < y_{i_0}\). On en déduit que \[x_{k+1} \leq y_{k+1}\]

Si \(i_0 = k+1\), alors \(x_{k+1} < y_{k+1}\), soit \[\sum_{i=1}^{k} y_i + y_{k+1} - \left( \sum_{i=1}^{k} x_i + x_{k+1} \right) \in \mathbb{P} \cup \{0\}\]

Donc, \[\sum_{i=1}^{k+1} y_i > \sum_{i=1}^{k+1} x_i\]

(d)

Pour \(n = 1\), \(x_1 \leq y_1\) et \(x_1 = y_1\) alors \(x_1 = y_1\). Pour \(n = 2\), si \(x_1 \leq y_1\) et \(x_2 \leq y_2\), alors \(x_1 + x_2 = y_1 + y_2\) implique \(x_1 = y_1\) et \(x_2 = y_2\). Donc \(P(2)\) est vraie.

Supposons que \(P(k)\) est vraie et montrons que \(P(k+1)\) est vraie.

Supposons que \(\forall i \in \{1, 2, \ldots, k+1\}, x_i \leq y_i\) et \(\sum_{i=1}^{k+1} x_i = \sum_{i=1}^{k+1} y_i\). On en déduit que \(x_{k+1} \leq y_{k+1}\), soit \[\sum_{i=1}^{k} y_i - \sum_{i=1}^{k} x_i \in \mathbb{P} \cup \{0\}\]

Ainsi, \[\sum_{i=1}^{k+1} y_i - \sum_{i=1}^{k+1} x_i = 0\]

Par conséquent, \(x_i = y_i\) pour \(i = 1, 2, \ldots, k+1\).

Exercice 2

  1. Montrons par récurrence que pour des nombres réels \(x_1, x_2, \ldots, x_n, y_1, y_2, \ldots, y_n\) nous avons les propriétés suivantes :

  1. \(\left| \sum_{i=1}^{n} x_i \right| \leq \sum_{i=1}^{n} |x_i|\).

  2. \(\forall i \in \{1, 2, \ldots, n\}, 0 \leq |x_i| \leq y_i\) alors \(\left| \prod_{i=1} x_i \right| \leq \prod_{i=1} |y_i|\).

  3. Soit \(n\) un entier naturel. Montrons que pour deux nombres réels \(x\) et \(y\) tels que \(0 < x < y\) alors \(x^n \leq y^n\).

Corrigé

(a)

Posons \(P(n)\) l’assertion \(\left| \sum_{i=1}^{n} x_i \right| \leq \sum_{i=1}^{n} |x_i|\).

  • Initialement, montrons que \(P(1)\) est vraie : On a \(|x_1| = |x_1|\) donc \(|x_1| - |x_1| \in \{0\} \subseteq \mathbb{P} \cup \{0\}\) donc \(|x_1| \leq |x_1|\). D’où \(P(1)\) est vraie.
  • Montrons que \(P(2)\) est vraie, c’est-à-dire \(|x_1 + x_2| \leq |x_1| + |x_2|\) par disjonction des cas. Remarquons que l’on peut toujours supposer que \(x_1 \leq x_2\).

Cas 1 : \(0 \leq x_1 \leq x_2\) et dans ce cas \(|x_1 + x_2| = x_1 + x_2 = |x_1| + |x_2|\). \(P(2)\) est vraie.

Cas 2 : \(x_1 > x_2 > 0\) et dans ce cas \(|x_1 + x_2| = -(x_1 + x_2) = -x_1 + (-x_2) = |x_1| + |x_2|\). \(P(2)\) est vraie.

Cas 3 : \(x_1 \leq 0 \leq x_2\) avec \(x_1 + x_2 \leq 0\) et \(|x_1 + x_2| = -(x_1 + x_2) = (-x_1) - x_2 = |x_1| + |x_2|\). \(P(2)\) est vraie dans ce cas.

Cas 4 : \(x_1 \leq 0 \leq x_2\) avec \(x_1 + x_2 > 0\) et \(|x_1 + x_2| = x_1 + x_2 = -|x_1| + |x_2|\). \(P(2)\) est vraie dans ce cas aussi. Nous avons montré que dans les 4 cas envisageables, donc \(P(2)\) est vraie.

  • Supposons que \(P(k)\) est vraie et montrons \(P(k+1)\) est vraie. On a \[\left| \sum_{i=1}^{k+1} x_i \right| = \left| \sum_{i=1}^{k} x_i + x_{k+1} \right| \leq \left| \sum_{i=1}^{k} x_i \right| + |x_{k+1}| \leq \sum_{i=1}^{k} |x_i| + |x_{k+1}| = \sum_{i=1}^{k+1} |x_i|\]

(b)

Posons \(P(n)\) l’assertion \(\left| \prod_{i=1} x_i \right| \leq \prod_{i=1} |y_i|\).

  • Initialement, on a \(0 \leq |x_1| \leq y_1\) alors \(|x_1| \leq |y_1|\). \(P(1)\) est vraie.
  • Pour \(n=2\), supposons que \(\forall i \in \{1, 2\}, 0 \leq |x_i| \leq y_i\). Nous avons

\[y_1 - |x_1| \in \mathbb{P} \cup \{0\}\] \[|x_2| \in \mathbb{P} \cup \{0\} \Rightarrow y_1|x_2| - |x_1x_2| \in \mathbb{P} \cup \{0\}\]

\[y_2 - |x_2| \in \mathbb{P} \cup \{0\}\] \[y_1 \in \mathbb{P} \cup \{0\} \Rightarrow y_1y_2 - y_1|x_2| \in \mathbb{P} \cup \{0\}\]

Les relations (1) et (2) donnent \[y_1|x_2| - |x_1||x_2| \in \mathbb{P} \cup \{0\}\] \[y_1y_2 - y_1|x_2| \in \mathbb{P} \cup \{0\}\] \[\Rightarrow y_1y_2 - |x_1x_2| \in \mathbb{P} \cup \{0\}\] \[i.e \ |x_1x_2| \leq y_1y_2 = |y_1||y_2|\]

  • Supposons que \(P(k)\) est vraie et montrons \(P(n + 1)\) est vraie. Supposons de plus que \(\forall i \in \{1, 2, \ldots, n+1\}, 0 \leq |x_i| \leq y_i\). Nous avons

\[y_{n+1} - |x_{n+1}| \in \mathbb{P} \cup \{0\}\] \[\left| \prod_{i=1}^{n} x_i \right| \in \mathbb{P} \cup \{0\} \Rightarrow y_{n+1} \left| \prod_{i=1}^{n} x_i \right| - \left| \prod_{i=1}^{n+1} x_i \right| \in \mathbb{P} \cup \{0\}\]

\[\prod_{i=1}^{n} |y_i| - \left| \prod_{i=1}^{n} x_i \right| \in \mathbb{P} \cup \{0\}\] \[y_{n+1} \in \mathbb{P} \cup \{0\} \Rightarrow y_{n+1} \prod_{i=1}^{n} |y_i| - y_{n+1} \left| \prod_{i=1}^{n} x_i \right| \in \mathbb{P} \cup \{0\}\]

Les relations (*) et (**) donnent \[y_{n+1} \left| \prod_{i=1} x_i \right| - \left| \prod_{i=1}^{n+1} x_i \right| \in \mathbb{P} \cup \{0\}\] \[y_{n+1} \prod_{i=1} |y_i| - y_{n+1} \left| \prod_{i=1} x_i \right| \in \mathbb{P} \cup \{0\}\] \[\Rightarrow \prod_{i=1}^{n+1} |y_i| - \left| \prod_{i=1}^{n+1} x_i \right| \in \mathbb{P} \cup \{0\}\] \[i.e \ | \prod_{i=1}^{n+1} x_i | \leq \prod_{i=1}^{n+1} |y_i|\]

(c)

Initialement, le cas \(n=1\) est trivial. Pour \(n=2\), supposons que \(0 \leq x \leq y\). Nous avons \[y - x \in \mathbb{P} \cup \{0\}\] \[x \in \mathbb{P} \cup \{0\} \Rightarrow xy - x^2 \in \mathbb{P} \cup \{0\}\] \[y - x \in \mathbb{P} \cup \{0\}\] \[y \in \mathbb{P} \cup \{0\} \Rightarrow y^2 - xy \in \mathbb{P} \cup \{0\}\]

De ces deux relations, \[(xy - x^2) + (y^2 - xy) \in \mathbb{P} \cup \{0\}\] \[i.e \ y^2 - x^2 \in \mathbb{P} \cup \{0\}\] \[x^2 \leq y^2\]

Supposons que \(P(k)\) est vraie et montrons que \(P(n + 1)\) est vraie. \[yn - xn \in \mathbb{P} \cup \{0\}\] \[x \in \mathbb{P} \cup \{0\} \Rightarrow xyn - x^{n+1} \in \mathbb{P} \cup \{0\}\] \[y - x \in \mathbb{P} \cup \{0\}\] \[yn \in \mathbb{P} \cup \{0\} \Rightarrow y^{n+1} - xyn \in \mathbb{P} \cup \{0\}\]

De ces deux relations, \[(xyn - x^{n+1}) + (y^{n+1} - xyn) \in \mathbb{P} \cup \{0\}\] \[i.e \ y^{n+1} - x^{n+1} \in \mathbb{P} \cup \{0\}\] \[x^{n+1} \leq y^{n+1}\]

Exercice 3

Soit \(E = \left\{ \frac{1}{n} + \frac{1}{m} \mid (n,m) \in \mathbb{N}^* \times \mathbb{N}^* \right\}\). Déterminer, si elles existent, la borne supérieure et la borne inférieure de \(E\).

Corrigé

Solution 1 :

Déterminons, si elles existent, la borne supérieure et la borne inférieure de \(E\).

Recherche de la borne supérieure :

Il est clair que \(E\) est non vide et notons par \(e_{n,m}\) les éléments de \(E\). \(\forall (n,m) \in \mathbb{N}^* \times \mathbb{N}^*\), nous avons \(n \geq 1\) et \(m \geq 1\). Par conséquent, \[\frac{1}{n} \leq 1 \quad \text{et} \quad \frac{1}{m} \leq 1\] \[\frac{1}{n} + \frac{1}{m} \leq 1 + 1 \Rightarrow e_{n,m} \leq e_{1,1}\]

On en déduit que \(e_{1,1}\) est un majorant de \(E\) ; de plus comme \(e_{1,1} \in E\) alors \[e_{1,1} = \max(E) = \sup(E)\]

Recherche de la borne inférieure :

On a \(\forall (n,m) \in \mathbb{N}^* \times \mathbb{N}^*\), \(0 < \frac{1}{n} + \frac{1}{m}\). Donc \(0\) est un minorant de \(E\).

Montrons que \(0 = \inf(E)\). Il suffit de montrer que \(y \in \mathbb{R}\), \(y > 0\), alors il existe \(\alpha \in E\) tel que \(0 < \alpha < y\). D’après la propriété d’Archimède, \[\forall y \in \mathbb{R}^*_+, \forall x \in \mathbb{R}^*_+, \exists n \in \mathbb{N}^* : ny > x\]

En posant \(x = e_{1,1}\), on obtient l’existence d’un entier naturel non nul \(n\) tel que \(ny > e_{1,1}\), autrement dit, tel que \(y > \frac{1}{n} + \frac{1}{n}\). Comme \(\alpha = e_{n,n} \in E\), on conclut que \(0\) est bien la borne inférieure de \(E\).

Exercice 4

Soit \(E\) la partie de \(\mathbb{R}\) définie par : \[E = \left\{ u_n = \frac{n - 1}{n} \cdot \frac{n + 1}{n} ; n \in \mathbb{N}^* \right\}.\]

  1. Montrons que \(E\) est infini et borné.

  2. Déterminer, si elles existent, la borne supérieure et la borne inférieure de \(E\).

Corrigé

1. Montrons que \(E\) est infini et borné :

Montrons que les éléments de \(E\) sont tous distincts, ce qui prouvera que \(E\) est infini. Raisonnons par l’absurde en supposant deux éléments égaux pour deux entiers strictement positifs \(n_1\) et \(n_2\) (avec \(n_1 \neq n_2\)) : \[u_{n_1} = u_{n_2} \Leftrightarrow \frac{n_1 - 1}{n_1} \cdot \frac{n_1 + 1}{n_1} = \frac{n_2 - 1}{n_2} \cdot \frac{n_2 + 1}{n_2} \Leftrightarrow n_1^2 = n_2^2 \Leftrightarrow n_1 = n_2\] Ce qui est absurde. Donc \(E\) est infini.

Montrons que \(u_n\) est borné. Pour tout \(n \in \mathbb{N}^*\), \(\frac{n - 1}{n} \leq 1\) et \(\frac{n + 1}{n} \leq 2\), donc \(u_n \leq 1\). De plus, pour tout \(n \in \mathbb{N}^*\), \(n > 1\) et \(\frac{n - 1}{n} > 0\), donc \(u_n > 0\). On en tire que \(E\) est minorée par \(0\) et majorée par \(1\).

2. Déterminer si elle existe la borne supérieure et la borne inférieure de \(E\) :

Sachant \(0 = u_1 \in E\), \(E\) admet une borne inférieure \(m\) et une borne supérieure \(M\) qui vérifient : \(m \geq 0\) et \(M \leq 1\).

Comme \(u_1 = 0\), on a \(\inf(E) = 0\).

Montrons que \(\sup(E) = 1\). Pour cela, pour tout \(\epsilon > 0\), montrons que \(1 - \epsilon\) n’est pas un majorant de \(E\). Il faut trouver \(u_n \in E\) tel que \(1 - \epsilon < u_n\). On a \[1 - \epsilon < u_n \Leftrightarrow 1 - \epsilon < \frac{n - 1}{n} \cdot \frac{n + 1}{n} \Leftrightarrow (1 - \epsilon) (n^2 + 1) < n^2 - 1 \Leftrightarrow n^2 > \frac{2 - \epsilon}{\epsilon}\]

Si \(\epsilon > 1\), on a \(1 - \epsilon < 0 \leq u_n\). Si \(\epsilon \leq 1\) alors \(\frac{2 - \epsilon}{\epsilon} = 1 + \frac{1 - \epsilon}{\epsilon} > 0\), on peut prendre \(n > \sqrt{\frac{2 - \epsilon}{\epsilon}}\) pour montrer que \(1 - \epsilon\) n’est pas un majorant de \(E\). Donc, on a \(\sup(E) = 1\).

Exercice 5

Soient \(x\) et \(y\) deux nombres réels.

  1. Montrons que \(|x| = \max\{x, -x\}\).

  2. Montrons que : \[\max\{x, y\} = \frac{x + y + |x - y|}{2}\]

Corrigé

1. Montrons que \(|x| = \max\{x, -x\}\) :

Par disjonction de cas selon le signe de \(x\) : - Si \(x \geq 0\) alors \(-x \leq 0\) donc \(|x| = x = \max\{x, -x\}\). - Si \(x \leq 0\) alors \(-x \geq 0\) donc \(|x| = -x = \max\{x, -x\}\).

On a démontré que dans les deux cas envisageables selon les signes de \(x\), la relation était vraie. Elle est donc toujours vraie.

2. Montrons que \(\max\{x, y\} = \frac{x + y + |x - y|}{2}\) :

Deux cas sont possibles : ou bien \(x \leq y\) ou bien \(x > y\).

Cas 1 : Si \(x \leq y\) alors \(x - y \leq 0\) et \[\max\{x, y\} = y \quad \text{et} \quad \frac{x + y + |x - y|}{2} = \frac{x + y - (x - y)}{2} = y\] \[\Rightarrow \max\{x, y\} = \frac{x + y + |x - y|}{2}\]

Cas 2 : Si \(x > y\) alors \(x - y > 0\) et \[\max\{x, y\} = x \quad \text{et} \quad \frac{x + y + |x - y|}{2} = \frac{x + y + (x - y)}{2} = x\] \[\Rightarrow \max\{x, y\} = \frac{x + y + |x - y|}{2}\]

On a démontré que dans les deux cas envisageables, la relation est vraie.

Exercice 6

Soit \(E\) la partie de \(\mathbb{R}\) définie par : \(E = \{ x \in \mathbb{Q}^+ \mid x^2 < 2 \}\).

  1. Montrons que \(E\) est majorée dans \(\mathbb{Q}\).

  2. Montrons que si \(a \in E\), il existe \(r \in \mathbb{Q}^+\) tel que \(a + r \in E\).

  3. \(E\) a-t-elle une borne supérieure dans \(\mathbb{Q}\) ?

Corrigé

1. Montrons que \(E\) est majorée dans \(\mathbb{Q}\) :

Soit \(x \in E\). On remarque que si \(x > 2\) alors \(x^2 > 4 > 2\) implique que \(x \notin E\), ce qui est une contradiction. On en déduit que pour tout \(x \in E\), \(x < 2\). De plus, pour tout \(x \in E\), on a \(x < 2\).

2. Montrons que si \(a \in E\), il existe \(r \in \mathbb{Q}^+\) tel que $a + r E :

Pour tout \(a \in E\), par définition on a \(a^2 < 2\), donc \(2 - a^2 > 0\). Posons \(r = \frac{2 - a^2}{2a + 2} \in \mathbb{Q}^+\).

On obtient \(r < \frac{2}{2a + 2} < 2\) et sachant que \(a \in \mathbb{Q}^+\), alors \(a + r \in \mathbb{Q}^+\). On obtient : \[(a + r)^2 = a^2 + 2ar + r^2 = a^2 + (2a + r)r < a^2 + (2a + 2)r = a^2 + 2 - a^2 = 2\] Donc \(a + r \in E\).

3. \(E\) a-t-elle une borne supérieure dans \(\mathbb{Q}\) :

Supposons qu’il existe \(y \in \mathbb{Q}\) tel que \(y = \sup(E)\). Donc \(1 \leq y \leq 2\) car \(1 \in E\) et \(2\) est un majorant de \(E\).

Cas 1 : Supposons que \(y^2 < 2\). On a \(y \in E\). D’après la question précédente, \(z = y + r \in E\). On a donc trouvé un \(z \in E\) tel que \(z > y = \sup(E)\). Ce qui est une contradiction.

Cas 2 : Supposons que \(y^2 > 2\). Soit \(\epsilon > 0\) tel que \((y - \epsilon)^2 > 2\). On a \((y - \epsilon)^2 = y^2 - 2y\epsilon + \epsilon^2\). Sachant que \(y \leq 2\), nous avons \(-2y\epsilon \geq -4\epsilon\) et que \[(y - \epsilon)^2 > y^2 - 4\epsilon + \epsilon^2 > y^2 - 4\epsilon\]

Si \(y^2 - 4\epsilon > 2\), c’est-à-dire \(\epsilon < \frac{y^2 - 2}{4}\) alors \((y - \epsilon)^2 > 2\).

Posons \(\epsilon = \frac{y^2 - 2}{6}\). On obtient \(\epsilon = \frac{y^2 - 2}{6} < \frac{2^2 - 2}{6} = \frac{1}{3} < 1 < y\).

Donc \(M = y - \frac{y^2 - 2}{6} \in \mathbb{Q}^+\), \(M < y\) et \(M^2 > 2\).

Supposons qu’il existe \(x \in E\) tel que \(x > M\), alors \(x^2 > M^2 > 2\), ce qui est absurde. Donc pour tout \(x \in E\), \(x < M\) et \(M\) est un majorant plus petit que \(y\). Ce qui est une contradiction.

Cas 3 : Supposons que \(y^2 = 2\). Sachant que \(y \in \mathbb{Q}^+\), il existe deux entiers naturels \(m\) et \(n\) premiers entre eux tels que \(y = \frac{m}{n}\). \(y^2 = 2\) implique que \[\left( \frac{m}{n} \right)^2 = 2 \Rightarrow m^2 = 2n^2 \Rightarrow m \text{ est pair, donc divisible par 2} \Rightarrow \frac{m}{2} \in \mathbb{Z}\] \[4 \left( \frac{m}{2} \right)^2 = 2n^2 \Rightarrow n^2 = 2 \left( \frac{m}{2} \right)^2 \Rightarrow n \text{ est pair, donc divisible par 2}\]

\(2\) est un facteur commun de \(m\) et \(n\), ce qui est absurde.

Exercice 7

Définition : Un sous-ensemble \(V\) de \(\mathbb{R}\) est appelé ouvert si pour tout réel \(x \in V\), il existe un intervalle ouvert \(]a, b[\) tel que \(x \in ]a, b[\) et \(]a, b[ \subseteq V\).

  1. Montrons qu’un ensemble \(V\) est un ouvert de \(\mathbb{R}\) si et seulement si pour tout \(a \in V\), il existe \(\epsilon > 0\) tel que \(I(a, \epsilon) \subseteq V\).
Corrigé

1. Montrons qu’un ensemble \(V\) est un ouvert de \(\mathbb{R}\) si et seulement si pour tout \(a \in V\), il existe \(\epsilon > 0\) tel que \(I(a, \epsilon) \subseteq V\) :

  • Supposons que \(V\) est un ouvert de \(\mathbb{R}\) et \(a \in V\). Alors il existe un intervalle \(]x, y[\) (ouvert) tel que \(a \in ]x, y[\) et \(]x, y[ \subseteq V\). Posons \(\epsilon = \min\{a - x; y - a\}\). On obtient : \[\epsilon \leq a - x \Rightarrow x \leq a - \epsilon < a < a + \epsilon\] \[\epsilon \leq y - a \Rightarrow a + \epsilon \leq y\] On en déduit que \[x \leq a - \epsilon < a < a + \epsilon \leq y \Rightarrow I(a, \epsilon) = ]a - \epsilon, a + \epsilon[ \subseteq ]x, y[ \subseteq V\]

    Ceci montre qu’il existe \(\epsilon > 0\) tel que \(I(a, \epsilon) \subseteq V\).

  • Supposons que pour tout \(a \in V\), il existe \(\epsilon > 0\) tel que \(I(a, \epsilon) \subseteq V\). Sachant que \(I(a, \epsilon) = ]a - \epsilon, a + \epsilon[\) est un intervalle ouvert contenant \(a\), par définition, \(V\) est un ouvert de \(\mathbb{R}\).
